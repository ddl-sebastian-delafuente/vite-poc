## Popover
Shows detailed information or actions on hover. It provides the option for richer content and more interactivity than a tooltip.

### Usage Guidelines
Use a popover
* To progressively disclose rich content or simple actions and show them in context.
* To add interactivity:
  * While maintaining the context of the parent / click target.
  * Without breaking the flow happening on a page.

Do not use a popover when:
  * If the content is only short text, consider using a <a href={bookMarks.TOOLTIP}>tooltip</a>.
  * If the content is longer or requires extended interactivity (like filling out a form) consider using
  an <a href={bookMarks.ACCORDION}>accordion</a> or <a href={bookMarks.MODAL}>modal</a>.
