### Usage Guidelines
Use tabs to reduce the complexity of navigation and/or content:
* <b>Navigation:</b> Tabs simplify the navigation above them. By delegating IA hierarchy down to tabs, the navigation is able to
display fewer choices that are easier to understand and work with. It is almost always more desireable to add tabs than
to add additional pages to the nav bar.
* <b>Content:</b> The organize the content below them. Tabs reduce the information on the page, providing a user with choices
that are often easier to make than if all content on a page were visible at once.

Consider whether there are situations (like an error, alert, or required task) where a particular tab may need to be
called out with a <a href={bookMarks.BADGE}>badge</a>.
