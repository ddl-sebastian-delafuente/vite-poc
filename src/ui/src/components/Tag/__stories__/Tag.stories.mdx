## Tag
A tag associates an object with 1 or more categories. It allows a user to view, apply, and remove categories for an object.

export const Template = ({text, closable, ...args}) => <Tag {...args} closable={closable}>{text}</Tag>

### Usage Guidelines
Tags should be used any time categories or labels can be present on an object. This can happen in tables, lists, and
any content container.
