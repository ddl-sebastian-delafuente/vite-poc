### Usage Guidelines
Because badges draw the eye, and because they lose effectiveness when there are too many, they should be used sparingly. They should be used only to mark things that you have high certainty a user would always want to investigate. If you wouldn’t send the user an email about the item being marked, think twice about whether it warrants a badge.

### Accessibility
Their small size and reliance on color for visual weight both raise accessibility concerns. This requires further investigation.
