### Usage Guidelines
Use a tree component on its own or use it inside a container like a modal or a select menu. Items in the tree can use any icons to represent that item. For collapsing and expanding the group, the caret icon and the relationship lines are standard and not to be modified.

Actions
* An item in the tree can be selected by clicking on the item or selecting a checkbox.
* Groups can be collapsed and expanded to hide or see the nested items.

### Accessibility
* Collapsing and expanding a group should be keyboard accessible.
* Selecting an item should be keyboard accessible.
